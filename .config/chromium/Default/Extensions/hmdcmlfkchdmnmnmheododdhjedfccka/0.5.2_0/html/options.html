Keyboard Shortcuts for Pick Color from Web Page

Pickup Color: Alt+P

Alt+P is default shortcut. Go to Extension page (chrome://extensions) to change it. There is link for changing shortcuts at bottom.

Cancel Pick: Esc

Simply cancel active picking color from web page. No color will be stored in history.

Update Colors: U

Eye dropper shows you only color which was present at moment of color pick active. If any javascript event occures (hover effect, menu appears) you can refresh its "cache" with this keyboard shortcut.
